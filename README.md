# 📝 Simple To-Do App

A **simple to-do list application** built with HTML, CSS, and JavaScript.  

---

## ✨ Features
- ✅ Add, complete, and delete tasks
- 💾 Data persists using `localStorage`
- 🌙 Dark theme with 3D effects
- 🖱️ Click tasks to mark them as completed
